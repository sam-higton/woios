The Story

Hello, and welcome to 'The shooter wot I made'. It's the first in a series of simple shooters that I'm making in attempt to teach myself games programming. It's in an engine that I'm putting together myself, just to see if I can. Feel free to browse the code on github, and be as critical as you like.

If you just so happen to be using this in Chrome, it should also work with a gamepad too. I tested with an Xbox 360 controller, but in theory, any x-input device should work.

note: By no means view this code as best practice, a lot of it I'm making up on the spot, and is probably woefully inefficient. If you're looking to learn games programming, I strongly recommend following the tutorials at tutsplus.

I'm also using the following libraries:

  • Raphael.js for rendering. This was probably a bad call, as it uses SVG over canvas, and canvas is much more efficient at rendering lots of objects, a bit like in a video game.
  • I'm using buzz.js to manage sounds. It's based around html5 audio, and I must say, it's pretty damn good and simple to use.
  • I'm also using blxr.net to generate my sound effects; a delighful little web based midi synthesizer.

So, enjoy the game, and feel free to contact me via twitter or facebook if you have any questions.
